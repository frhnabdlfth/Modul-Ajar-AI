# Modul Ajar AI

## Project Documentation

### Overview
Modul Ajar AI is an advanced AI-powered learning module designed to facilitate interactive learning experiences. It utilizes machine learning algorithms to adapt to the learner's pace and understanding, providing personalized content and assessments.

### Features
- **Adaptive Learning**: AI-driven adjustments to content based on user performance.
- **Intuitive User Interface**: Designed for ease of use, catering to all age groups.
- **Comprehensive Analytics**: Provides insightful metrics and reports to monitor learning progress.
- **Multi-language Support**: Enables learning in various languages.

## Setup Instructions

### Prerequisites
- Python 3.x installed
- Required libraries (listed in `requirements.txt`) installed

### Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/frhnabdlfth/Modul-Ajar-AI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Modul-Ajar-AI
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```

### Usage
- Open your web browser and go to `http://localhost:5000` to access the application.  
- Follow the on-screen instructions to get started with interactive learning.
